# Official Brand Asset Blocker

Official Youtoola logo files are not present in the repository.

Before Phase 2 begins, the owner must provide and approve:

- primary logo
- compact or icon mark, if one exists
- light and dark variants
- SVG source where available
- usage and clear-space guidance
- confirmation that Youtoola may publish the assets

Codex must not redraw, reinterpret, or generate a replacement logo. Phase 1 uses text only and does not attempt brand-system implementation.
