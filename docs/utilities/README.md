# Utility Specifications

Each approved utility receives a reviewed specification here before BUILD. A Sheet record or `idea` registry entry is not a specification and does not authorize a route or calculator implementation.

A specification must record its source coordinates and hash, search and competitor research, commercial scorecard, calculation contract, assumptions, validation, accessibility, SEO, related-tool and measurement requirements. Released source changes require owner impact review.

Fuel Trip Calculator remains unreleased in the registry. Its RAPID implementation candidate may be reviewed only through a private, noindexed Preview until Growth Infrastructure and SHIP gates are complete.
